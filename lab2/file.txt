Утилита cat:

cat (сокращение от "concatenate") — это стандартная утилита UNIX/Linux, которая используется для вывода содержимого одного или нескольких файлов на экран или объединения их содержимого в новый файл.

Основные функции:

    Вывод содержимого файла на стандартный вывод (экран).
    Объединение нескольких файлов в один.
    Создание нового файла с введённым содержимым (при использовании перенаправления).

Примеры:

    cat file.txt — выводит содержимое файла file.txt.
    cat file1.txt file2.txt > combined.txt — объединяет содержимое двух файлов и записывает его в файл combined.txt.
    cat -n file.txt — выводит содержимое файла с нумерацией строк.

Утилита grep:

grep (сокращение от "global regular expression print") — это мощная утилита для поиска строк, соответствующих заданному шаблону (регулярному выражению), в файлах или на стандартном вводе.

Основные функции:

    Поиск текста в файле или на стандартном вводе.
    Фильтрация строк, соответствующих или не соответствующих шаблону.
    Поддержка регулярных выражений для сложных поисков.

Примеры:

    grep pattern file.txt — ищет строки, содержащие pattern в файле file.txt.
    cat file.txt | grep pattern — ищет строки, содержащие pattern, в выводе команды cat.
    grep -i pattern file.txt — ищет строки, игнорируя регистр.
    grep -v pattern file.txt — выводит строки, которые не содержат pattern.

grep часто используется в конвейерах (пайпах) для фильтрации вывода других команд.
